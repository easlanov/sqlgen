# Magritte Static Page

Простой пример статичной страницы на React + Magritte.

## Установка

```bash
cd magritte-static-page
yarn install
```

## Разработка

```bash
yarn dev
```

## Сборка статичной страницы

```bash
yarn build
```

После сборки будут созданы:

- `dist/index.html` — ссылается на скомпилированные JS и CSS
- `dist/assets/*.js` — JS-бандл
- `dist/assets/*.css` — CSS-бандл
